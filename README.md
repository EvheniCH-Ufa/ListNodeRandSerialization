# Сериализатор двусвязного списка с произвольными ссылками

## 📋 Описание проекта

Данная программа реализует сериализацию и десериализацию двусвязного списка `ListNode` с произвольными ссылками (`rand`). Проект выполнен в рамках тестового задания на C++.

### 📁 Структура данных

```cpp
struct ListNode {
    ListNode* prev = nullptr;   // указатель на предыдущий элемент
    ListNode* next = nullptr;   // указатель на следующий элемент
    ListNode* rand = nullptr;   // указатель на произвольный элемент списка
    std::string data;           // пользовательские данные
};
```

> **⚠️ Важно:** Структура `ListNode` является **неизменяемой** (модифицировать нельзя).

## ⚙️ Функциональность

- **Чтение текстового описания списка** из файла `inlet.in`
- **Построение двусвязного списка** по прочитанным данным
- **Сериализация списка** в бинарный файл `outlet.out`

## 📄 Формат входного файла

Входной файл `inlet.in` имеет текстовый формат, где каждая строка описывает один узел:

```
<data>;<rand_index>
```

| Поле | Описание |
|------|----------|
| `<data>` | строка произвольного содержания (может содержать пробелы, спецсимволы, UTF-8) |
| `<rand_index>` | индекс узла, на который указывает `rand`, либо `-1` (соответствует `nullptr`) |

### Пример входного файла

```
apple;2
banana;-1
carrot;1
```

В данном примере:
- **Узел 0:** `"apple"` → `rand` указывает на узел 2
- **Узел 1:** `"banana"` → `rand = nullptr`
- **Узел 2:** `"carrot"` → `rand` указывает на узел 1

## 📦 Формат выходного файла

Выходной файл `outlet.out` — **бинарный файл**, содержащий сериализованное представление списка.

Для каждого узла записываются:

| Поле | Тип | Описание |
|------|-----|----------|
| Длина строки | `size_t` | Размер строки data |
| Данные строки | `char[]` | Содержимое строки data |
| Индекс rand | `int` | Индекс узла, на который указывает rand (или -1) |

## 🔒 Ограничения

| Параметр | Значение |
|----------|----------|
| Максимальное количество узлов | **10⁶** |
| Максимальная длина строки data | **1000 символов** |
| Кодировка строк | **UTF-8** |

## 🛠️ Сборка проекта

Проект использует **CMake** (версия 3.5 или выше):

```bash
# Создание директории для сборки
mkdir build
cd build

# Генерация Makefile и сборка
cmake ..
make
```

После сборки будет создан исполняемый файл **`ListNodeRandSerialization`**.

## 💻 Использование

Программа принимает имена входного и выходного файлов через конструктор класса `Serializator`:

```cpp
#include "ListNodeSerializator.h"

int main() {
    Serializator ser("inlet.in", "outlet.out");
    return 0;
}
```

## 🏗️ Детали реализации

### Класс Serializator

Основной класс, отвечающий за сериализацию и десериализацию списка.

#### Поля класса:

| Поле | Описание |
|------|----------|
| `head_` | указатель на первый элемент списка |
| `links_` | хеш-таблица для сопоставления адресов узлов с их индексами |

#### Основные методы:

| Метод | Описание |
|-------|----------|
| `LoadData()` | чтение и парсинг входного файла |
| `SaveData()` | сохранение списка в бинарный файл |
| `ParceLine()` | парсинг одной строки входного файла |
| `MakeLinks()` | установка связей `rand` после создания всех узлов |
| `GetRoot()` | получение указателя на начало списка |

### Алгоритм работы

#### 1️⃣ Загрузка данных
#### 2️⃣ Установка связей rand
#### 3️⃣ Сериализация
#### 4️⃣ Очистка памяти

## 📊 Пример работы

### Входной файл (inlet.in)
```
Hello World;2
Test;-1
Example;0
```

### Построенный список

| Индекс | data | prev | next | rand |
|--------|------|------|------|------|
| 0 | "Hello World" | nullptr | узел 1 | узел 2 |
| 1 | "Test" | узел 0 | узел 2 | nullptr |
| 2 | "Example" | узел 1 | nullptr | узел 0 |

### Выходной бинарный файл (outlet.out)

```
[длина:11][Hello World][индекс:2]
[длина:4][Test][индекс:-1]
[длина:7][Example][индекс:0]
```

## 📝 Примечания

- Связи `prev` и `next` **не сохраняются** в бинарный файл, так как они должны быть восстановлены при десериализации
- Для восстановления списка потребуется реализовать метод десериализации
- Индексация узлов начинается с **0**
- При некорректном индексе `rand` связь не устанавливается
